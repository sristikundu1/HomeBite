# HomeBite Project Rules

## Tech Stack

Frontend

- React (Vite)
- JavaScript
- Tailwind CSS
- React Router
- Firebase Authentication
- Axios
- TanStack Query
- React Hook Form
- Zod
- Framer Motion
- Swiper

Backend

- Node.js
- Express
- MongoDB Native Driver
- Socket.IO
- Stripe

Never use TypeScript.

Never use Mongoose.

Never redesign existing UI.

Always reuse components.

Use Tailwind.

Use Framer Motion.

Use Swiper for sliders.

Use Lucide React icons.

Use react-icons only for brand icons.

Keep dark/light mode working.

Never break responsiveness.

Follow the folder structure described in docs.
